# Letterbox Hardware

## Rev 1

### Pin Layout

#### Servo

* GPIO 18
* -Ground
* +5V

#### LED

* GPIO 22
* -Ground
* +5V
* Resistor: 40 Ohm
* Array: Bourns 4116R-1-470LF (8 resistors)
* <http://www.conrad.com/ce/en/product/1055170/Bourns-4116R-1-470LF?ref=list>

#### Photoresistor

* GPIO 4
* -GROUND
* +3V

#### Powerswitch

* GPIO 17
* -GROUND
* +3V
